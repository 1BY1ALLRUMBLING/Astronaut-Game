Astronaut Game by 1by1

Game Rules

1. Control the astronaut using the arrow keys to move.

2. Use the spacebar to shoot.

3. Avoid collisions with asteroids and shoot them to destroy.

4. Collect power-ups for speed or invincibility boosts.

5. When the boss appears, destroy it to continue the game. Beware of its small asteroids!

6. The game ends if the astronaut loses all lives.

7. Score as many points as possible!
